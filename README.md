# BRK Solar Solution

A modern, responsive website for BRK Solar - solar panel installation and energy solutions.

## Features

- Modern, responsive design
- Solar panel installation services showcase
- System design and sizing information
- Working contact/quote request form
- FAQ section
- Complete process workflow
- Mobile-friendly navigation

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.
